# ✈️ Travelora Frontend

Travelora is a modern travel package booking platform built using React.js and Tailwind CSS. It provides a clean, responsive user interface for browsing travel packages, viewing details, and managing bookings.

---

## 🚀 Features

* Browse and explore travel packages
* View detailed package information
* Responsive and modern UI design
* Role-based UI support (User / Agent / Admin)
* Dynamic filtering and search functionality
* Booking interface with smooth user experience

---

## 🛠️ Tech Stack

* **Frontend:** React.js, JavaScript (ES6+)
* **Styling:** Tailwind CSS
* **API Handling:** Axios
* **State Management:** Context API
* **Tools:** Git, GitHub, VS Code

---

## 📂 Project Structure

```
src/
│
├── components/     # Reusable UI components
├── pages/          # Application pages (User, Agent, Admin)
├── context/        # Authentication & global state
├── services/       # API service functions
├── assets/         # Images & static files
└── App.jsx         # Main app entry
```

---

## 🔐 Key Functionalities

* Authentication UI with JWT integration
* Role-based routing (User / Agent / Admin)
* Package browsing and filtering
* Booking UI and history interface
* Responsive design for all devices

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/rohith-git2004/Travelora-Frontend.git
```

### 2. Navigate to project

```
cd Travelora-Frontend
```

### 3. Install dependencies

```
npm install
```

### 4. Run the project

```
npm run dev
```

---

## 🌐 Backend API

This frontend connects to a backend built with Node.js, Express.js, and MongoDB.
